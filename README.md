# Python Data Journey 🚀

This repository documents my **12-month sprint-based learning journey** to become a Python-focused data professional.  
The roadmap covers **Data Analytics → Backend Development → Data Science → AI/ML**, with projects designed to simulate **real freelance gigs** using Kaggle datasets.

---

## 📅 Roadmap
- **Month 1–3:** Data Analytics Foundations (pandas, cleaning, visualization, Excel automation, web scraping)
- **Month 4–6:** Backend Development for Data Systems (APIs, FastAPI, reporting automation)
- **Month 7–9:** Data Science (statistics, ML models, feature engineering)
- **Month 10–12:** AI/ML (neural networks, NLP, recommendation systems)

---

## 🏗 Sprint System
Each sprint follows a **3-day cycle**:
1. **Day 1 – Concept Learning**  
   Explanation, Python examples, exercises.
2. **Day 2 – Project Building**  
   Freelance-style project with Kaggle dataset.
3. **Day 3 – Review & Practice**  
   Debugging, reflection, GitHub documentation, LinkedIn/X post.

---

## 📂 Repository Structure
- `sprints/` → Learning sprints with exercises and projects.
- `freelance_projects/` → Polished, client-ready deliverables.
- `datasets/` → Kaggle datasets or dataset links.
- `README.md` → Overview of the journey.
